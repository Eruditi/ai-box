# AI边缘计算摄像头分析盒子 - 部署使用手册

## 📦 项目简介

即插即用的AI摄像头分析盒子系统，支持：
- ✅ USB摄像头自动检测
- ✅ IP摄像头（RTSP流）支持
- ✅ 实时物体检测（人物、车辆等）
- ✅ 人脸检测
- ✅ Web管理界面
- ✅ 实时视频流
- ✅ 系统自启动和自恢复

---

## 🛠️ 快速部署

### 方式一：一键安装（推荐）

```bash
# 1. 进入项目目录
cd /workspace/ai-box

# 2. 运行安装脚本
sudo bash scripts/install.sh

# 3. 下载预训练模型
sudo bash scripts/download_models.sh

# 4. 启动服务
sudo systemctl start ai-box
```

### 方式二：Docker部署

```bash
# 构建Docker镜像
cd /workspace/ai-box
docker build -t ai-box .

# 运行容器
docker run -d \
  --name ai-box \
  --privileged \
  --device=/dev/video0:/dev/video0 \
  -p 8080:8080 \
  -v /var/lib/ai-box/data:/var/lib/ai-box/data \
  -v /var/log/ai-box:/var/log/ai-box \
  ai-box
```

### 方式三：手动启动（开发调试）

```bash
cd /workspace/ai-box

# 安装依赖
pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt

# 下载模型
bash scripts/download_models.sh

# 启动程序
AI_BOX_CONFIG=config/settings.yaml python3 src/main.py
```

## 📖 使用说明

### 1. 硬件连接

#### USB摄像头
- 直接插入设备的USB接口
- 系统会自动检测并开始工作

#### IP摄像头（RTSP）
- 编辑配置文件：`/etc/ai-box/settings.yaml`
- 在 `camera.input_sources` 中添加RTSP地址：
  ```yaml
  camera:
    input_sources:
      - "rtsp://192.168.1.100:554/stream"
  ```
- 重启服务：`sudo systemctl restart ai-box`

### 2. 访问Web界面

- 在浏览器中打开：`http://<设备IP>:8080`
- 查看实时视频和检测结果
- 切换摄像头（如连接多个）

### 3. 常用命令

```bash
# 启动服务
sudo systemctl start ai-box

# 停止服务
sudo systemctl stop ai-box

# 重启服务
sudo systemctl restart ai-box

# 查看状态
sudo systemctl status ai-box

# 查看实时日志
sudo journalctl -u ai-box -f

# 设置开机自启
sudo systemctl enable ai-box

# 取消开机自启
sudo systemctl disable ai-box
```

## ⚙️ 配置说明

配置文件位置：`/etc/ai-box/settings.yaml`

### 摄像头配置
```yaml
camera:
  auto_detect: true              # 自动检测USB摄像头
  default_resolution: "1920x1080"
  fps: 30
  input_sources:                 # 手动指定视频源
    - "/dev/video0"
    - "rtsp://192.168.1.100:554/stream"
```

### AI分析配置
```yaml
ai:
  enabled: true
  inference_engine: "opencv"
  model_path: "/opt/ai-box/models/"
  confidence_threshold: 0.5      # 置信度阈值
  analysis_mode: "object_detection"
```

### Web界面配置
```yaml
web:
  enabled: true
  host: "0.0.0.0"
  port: 8080
  auth_enabled: false
```

### 健康监控配置
```yaml
health:
  check_interval: 30             # 健康检查间隔（秒）
  auto_restart: true             # 自动重启
```

## 🔧 故障排除

### 问题：无法检测到USB摄像头
```bash
# 检查摄像头设备
ls -l /dev/video*

# 查看摄像头信息
v4l2-ctl --list-devices
```

### 问题：服务无法启动
```bash
# 查看详细日志
sudo journalctl -u ai-box -n 100 --no-pager

# 检查端口占用
sudo netstat -tlnp | grep 8080

# 手动测试运行
cd /opt/ai-box
sudo AI_BOX_CONFIG=/etc/ai-box/settings.yaml python3 src/main.py
```

### 问题：Web界面无法访问
```bash
# 检查防火墙
sudo ufw status
sudo ufw allow 8080

# 检查服务是否监听端口
sudo ss -tlnp | grep 8080
```
