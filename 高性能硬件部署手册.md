# 高性能AI边缘计算服务器部署手册

## 一、硬件规格要求

### 1.1 核心硬件配置

| 组件 | 规格要求 |
|------|----------|
| CPU | 高性能多核嵌入式CPU |
| 内存 | 64GB DDR4 |
| 系统存储 | 1TB SSD |
| GPU | 24GB以上显存 |
| RAID盘位 | 12-24盘位磁盘阵列 |
| AI运算单元 | 7-21个嵌入式AI运算单元 |

### 1.2 网络接口

- 4个100M/1000M自适应以太网接口
- 支持三种网络模式：
  - 多址设定（Multi Address）
  - 负载均衡（Load Balancing）
  - 主备模式（Active Backup）
- RJ45接口
- 物联网接口：RS485、IO、UART

### 1.3 视频接入能力

- 支持协议：ONVIF、RTSP、GB28181、GB28181级联
- 最大接入能力：16*N路 1080P，最大336路
- 解码类型：H.264/H.265

## 二、系统安装

### 2.1 操作系统选择

推荐操作系统：
- Ubuntu Server 20.04 LTS 或更高版本
- CentOS 8 Stream 或更高版本
- Debian 11 或更高版本

### 2.2 系统安装步骤

1. 制作启动盘
   ```bash
   # 使用dd命令制作启动盘（Linux）
   sudo dd if=ubuntu-22.04-live-server-amd64.iso of=/dev/sdX bs=4M status=progress
   ```

2. BIOS/UEFI设置
   - 启用虚拟化支持（VT-x/AMD-V）
   - 设置从U盘启动
   - 禁用Secure Boot（如需安装第三方驱动）

3. 系统安装
   - 选择语言和时区
   - 配置网络（至少一个接口）
   - 创建用户和密码
   - 分区方案：
     - /boot：1GB
     - /：剩余空间（使用LVM便于扩展）

### 2.3 基础系统配置

1. 更新系统
   ```bash
   sudo apt update && sudo apt upgrade -y
   ```

2. 安装必备工具
   ```bash
   sudo apt install -y vim git curl wget net-tools htop iotop
   ```

3. 配置时区
   ```bash
   sudo timedatectl set-timezone Asia/Shanghai
   ```

## 三、RAID存储配置

### 3.1 RAID级别选择

| RAID级别 | 特点 | 推荐用途 |
|----------|------|----------|
| RAID 0 | 高性能，无冗余 | 临时存储、缓存 |
| RAID 1 | 高冗余，低性能 | 系统盘 |
| RAID 5 | 平衡性能和冗余 | 通用存储 |
| RAID 6 | 高冗余，双校验 | 视频存储（推荐） |
| RAID 10 | 高性能+高冗余 | 高性能应用 |

### 3.2 软件RAID配置（mdadm）

1. 安装mdadm
   ```bash
   sudo apt install -y mdadm
   ```

2. 创建RAID 6阵列（示例）
   ```bash
   # 假设使用8块硬盘 /dev/sdb - /dev/sdi
   sudo mdadm --create --verbose /dev/md0 --level=6 --raid-devices=8 /dev/sdb /dev/sdc /dev/sdd /dev/sde /dev/sdf /dev/sdg /dev/sdh /dev/sdi
   ```

3. 查看RAID状态
   ```bash
   cat /proc/mdstat
   sudo mdadm --detail /dev/md0
   ```

4. 格式化和挂载
   ```bash
   # 格式化为ext4
   sudo mkfs.ext4 /dev/md0
   
   # 创建挂载点
   sudo mkdir -p /data/video
   
   # 临时挂载
   sudo mount /dev/md0 /data/video
   
   # 永久挂载（编辑/etc/fstab）
   echo '/dev/md0 /data/video ext4 defaults 0 0' | sudo tee -a /etc/fstab
   ```

5. 保存RAID配置
   ```bash
   sudo mdadm --detail --scan | sudo tee -a /etc/mdadm/mdadm.conf
   sudo update-initramfs -u
   ```

### 3.3 存储目录结构

```
/data/
├── video/              # 视频存储
│   ├── hot/           # 热数据（最近7天）
│   ├── warm/          # 温数据（7-30天）
│   └── cold/          # 冷数据（30天以上）
├── ai_models/          # AI模型
├── logs/              # 日志文件
└── temp/              # 临时文件
```

创建目录结构：
```bash
sudo mkdir -p /data/video/{hot,warm,cold}
sudo mkdir -p /data/ai_models
sudo mkdir -p /data/logs
sudo mkdir -p /data/temp
sudo chown -R $USER:$USER /data/
```

## 四、网络配置

### 4.1 网络接口命名

查看网络接口：
```bash
ip link show
```

### 4.2 多址设定配置（Single Interface, Multiple IPs）

1. 使用ip命令添加IP别名
   ```bash
   # 为主接口eth0添加第二个IP
   sudo ip addr add 192.168.2.100/24 dev eth0
   
   # 查看配置
   ip addr show eth0
   ```

2. 使用netplan永久配置（Ubuntu）
   ```yaml
   # /etc/netplan/01-netcfg.yaml
   network:
     version: 2
     ethernets:
       eth0:
         addresses:
           - 192.168.1.100/24
           - 192.168.2.100/24
         routes:
           - to: default
             via: 192.168.1.1
         nameservers:
           addresses: [8.8.8.8, 114.114.114.114]
   ```
   
   应用配置：
   ```bash
   sudo netplan apply
   ```

### 4.3 负载均衡配置（Bonding Mode 0）

1. 加载bonding模块
   ```bash
   sudo modprobe bonding
   ```

2. 创建bonding接口
   ```bash
   # 添加bond0接口
   echo "+bond0" | sudo tee /sys/class/net/bonding_masters
   
   # 设置为负载均衡模式（balance-rr）
   echo 0 | sudo tee /sys/class/net/bond0/bonding/mode
   
   # 设置链路监测间隔
   echo 100 | sudo tee /sys/class/net/bond0/bonding/miimon
   ```

3. 添加从接口
   ```bash
   # 先关闭从接口
   sudo ip link set eth0 down
   sudo ip link set eth1 down
   sudo ip link set eth2 down
   sudo ip link set eth3 down
   
   # 添加到bond0
   echo "+eth0" | sudo tee /sys/class/net/bond0/bonding/slaves
   echo "+eth1" | sudo tee /sys/class/net/bond0/bonding/slaves
   echo "+eth2" | sudo tee /sys/class/net/bond0/bonding/slaves
   echo "+eth3" | sudo tee /sys/class/net/bond0/bonding/slaves
   
   # 启用bond0
   sudo ip link set bond0 up
   ```

4. 配置IP地址
   ```bash
   sudo ip addr add 192.168.1.100/24 dev bond0
   sudo ip route add default via 192.168.1.1
   ```

5. 永久配置（netplan）
   ```yaml
   # /etc/netplan/01-netcfg.yaml
   network:
     version: 2
     bonds:
       bond0:
         interfaces: [eth0, eth1, eth2, eth3]
         addresses: [192.168.1.100/24]
         routes:
           - to: default
             via: 192.168.1.1
         nameservers:
           addresses: [8.8.8.8, 114.114.114.114]
         parameters:
           mode: balance-rr
           mii-monitor-interval: 100
   ```

### 4.4 主备模式配置（Bonding Mode 1）

1. 创建bonding接口（主备模式）
   ```bash
   echo "+bond0" | sudo tee /sys/class/net/bonding_masters
   echo 1 | sudo tee /sys/class/net/bond0/bonding/mode
   echo 100 | sudo tee /sys/class/net/bond0/bonding/miimon
   
   # 设置主接口
   echo "eth0" | sudo tee /sys/class/net/bond0/bonding/primary
   ```

2. 添加从接口（同4.3步骤3）

3. 永久配置（netplan）
   ```yaml
   # /etc/netplan/01-netcfg.yaml
   network:
     version: 2
     bonds:
       bond0:
         interfaces: [eth0, eth1, eth2, eth3]
         addresses: [192.168.1.100/24]
         routes:
           - to: default
             via: 192.168.1.1
         nameservers:
           addresses: [8.8.8.8, 114.114.114.114]
         parameters:
           mode: active-backup
           primary: eth0
           mii-monitor-interval: 100
   ```

## 五、GPU驱动和CUDA安装

### 5.1 NVIDIA驱动安装

1. 禁用nouveau驱动
   ```bash
   echo "blacklist nouveau" | sudo tee /etc/modprobe.d/blacklist-nouveau.conf
   echo "options nouveau modeset=0" | sudo tee -a /etc/modprobe.d/blacklist-nouveau.conf
   sudo update-initramfs -u
   ```

2. 重启系统
   ```bash
   sudo reboot
   ```

3. 安装NVIDIA驱动
   ```bash
   sudo apt install -y nvidia-driver-535
   ```

4. 验证安装
   ```bash
   nvidia-smi
   ```

### 5.2 CUDA和cuDNN安装

1. 下载CUDA Toolkit
   ```bash
   wget https://developer.download.nvidia.com/compute/cuda/12.1.0/local_installers/cuda_12.1.0_530.30.02_linux.run
   sudo sh cuda_12.1.0_530.30.02_linux.run
   ```

2. 配置环境变量
   ```bash
   echo 'export PATH=/usr/local/cuda/bin:$PATH' >> ~/.bashrc
   echo 'export LD_LIBRARY_PATH=/usr/local/cuda/lib64:$LD_LIBRARY_PATH' >> ~/.bashrc
   source ~/.bashrc
   ```

3. 验证CUDA
   ```bash
   nvcc --version
   ```

4. 安装cuDNN（需要NVIDIA账号）
   - 下载cuDNN包
   - 解压并复制文件：
     ```bash
     sudo cp include/cudnn*.h /usr/local/cuda/include
     sudo cp lib/libcudnn* /usr/local/cuda/lib64
     sudo chmod a+r /usr/local/cuda/include/cudnn*.h /usr/local/cuda/lib64/libcudnn*
     ```

## 六、AI边缘计算系统部署

### 6.1 项目获取和依赖安装

1. 克隆项目
   ```bash
   cd /opt
   sudo git clone <repository-url> ai-box
   cd ai-box
   ```

2. 安装Python依赖
   ```bash
   # 使用国内镜像源
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

3. 安装系统依赖
   ```bash
   sudo apt install -y libgl1-mesa-glx libglib2.0-0
   ```

### 6.2 配置文件修改

1. 复制高性能配置文件
   ```bash
   cp config/settings_high_performance.yaml config/settings.yaml
   ```

2. 根据实际硬件修改配置
   ```yaml
   # config/settings.yaml
   storage:
     video_path: /data/video
     max_storage_gb: 10000  # 根据实际RAID容量调整
   ```

### 6.3 系统服务配置

1. 复制systemd服务文件
   ```bash
   sudo cp scripts/ai-box.service /etc/systemd/system/
   ```

2. 修改服务文件中的路径（如需要）
   ```ini
   [Service]
   WorkingDirectory=/opt/ai-box
   ExecStart=/usr/bin/python3 /opt/ai-box/src/main.py
   ```

3. 启用并启动服务
   ```bash
   sudo systemctl daemon-reload
   sudo systemctl enable ai-box
   sudo systemctl start ai-box
   ```

4. 查看服务状态
   ```bash
   sudo systemctl status ai-box
   sudo journalctl -u ai-box -f
   ```

## 九、性能优化建议

### 9.1 系统级优化

1. 调整文件描述符限制
   ```bash
   echo "* soft nofile 65536" | sudo tee -a /etc/security/limits.conf
   echo "* hard nofile 65536" | sudo tee -a /etc/security/limits.conf
   ```

2. 启用TCP BBR拥塞控制
   ```bash
   echo "net.core.default_qdisc=fq" | sudo tee -a /etc/sysctl.conf
   echo "net.ipv4.tcp_congestion_control=bbr" | sudo tee -a /etc/sysctl.conf
   sudo sysctl -p
   ```

### 9.2 应用级优化

1. 使用TensorRT加速推理
2. 启用GPU内存池
3. 调整批处理大小
4. 优化视频解码参数
